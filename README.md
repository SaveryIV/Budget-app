# 📖 [Recipe app] <a name="about-project"></a>

The **Budget App** is a web service where you can save all the transactions you have made in groups, and categorize them in a basic way, with the possibility of check how much money have you spend in total in that specific group.

## 🛠 Built With <a name="built-with"></a>

Ruby, Ruby on Rails, Postgresql, Visual Studia Code

### Prerequisites <a name="prerequisites"></a>

- Ruby
- Ruby on Rails
- PostgreSQL
- Git

### Install <a name="install"></a>

To install the project's dependencies, run:

```
bundle install
```

### Usage <a name="usage"></a>

To use, run:

```
rails server
or
rails s
```

### Run tests <a name="run tests"></a>

To test, run:

```
rspec
```
